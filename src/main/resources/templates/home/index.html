Functionality

The Job Post Board is a web application designed to facilitate job postings for developers and recruiters. It provides the following functionality:

  • User Authentication: Users can register, login, and logout securely to access the platform.
  • User Roles: The application supports two main user roles - DEVELOPER and RECRUITER.
  • Job Posting: Recruiters can create and publish job postings with details such as job title, description, etc.
  • Job Application: Developers can view job postings and apply to the ones they are interested in.
  • Dashboard: Each user has a personalized dashboard where they can manage their job postings (for recruiters) or applications (for developers).

Technologies Used

  • Java Spring
  • Spring Security
  • Spring MVC Architecture
  • PostgreSQL
  • HTML/CSS/JavaScript along with bootstrap
  • Thymeleaf
